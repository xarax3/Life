# نتیجه مورد انتظار

پس از پایان Day 3 باید بتوانید:

- تحلیل کامل Flow یک تیم را انجام دهید.
- گزارش‌های Control Chart و CFD را تفسیر کنید.
- گلوگاه‌های فرآیند را شناسایی کنید.
- از متریک‌های Flow برای بهبود فرآیند استفاده کنید.
- Dashboardهای مدیریتی برای Scrum، Kanban و DevOps طراحی کنید.
- در مصاحبه‌های Jira Administrator، Scrum Master و Agile Coach به سؤالات مرتبط با Flow Metrics پاسخ دهید.

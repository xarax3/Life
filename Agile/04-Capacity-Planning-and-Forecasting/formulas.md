# Day 4 - Formulas

تمام فرمول‌های مهم Capacity Planning، Forecasting و KPIهای Jira

---

# Capacity

```
Capacity

=

Available Working Hours

-

Leave

-

Meetings

-

Training

-

Support
```

---

# Team Capacity

```
Team Capacity

=

Sum(Member Capacities)
```

---

# Initial Commitment

```
Story Points

یا

Number of Issues

که در Sprint Planning Commit شده‌اند.
```

---

# Final Commitment

```
Completed Initial Story Points
```

---

# Commitment Reliability

```
Final Commitment

/

Initial Commitment

×

100
```

---

# Focus Factor

```
Planned Work

Available Capacity

100
```

Used Capacity

100

# Average Velocity

```
Sum(Velocity)

/

Number of Sprints
```

---

# Velocity Forecast

```
Remaining Story Points

/

Average Velocity
```

```
Estimated Sprints

×

Sprint Duration
```

---

# Resource Allocation

Assigned Hours

/

Available Hours

100

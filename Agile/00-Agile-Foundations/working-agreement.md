# Working Agreement و Definition of Done

## قرارداد کار حداقلی

| موضوع | توافق نمونه |
|---|---|
| آماده‌بودن Story | ارزش، پذیرش‌پذیری، Owner و برآورد روشن باشد |
| شروع کار | WIP limit رعایت شود؛ Blocker همان روز قابل‌دیدن شود |
| بازبینی | Pull request و معیار کیفیت قبل از Done تکمیل شوند |
| ارتباط | تغییر اولویت یا Scope در Board ثبت شود، نه فقط در پیام خصوصی |
| بهبود | یک اقدام Retrospective با Owner و موعد داشته باشد |

## Definition of Done نمونه

```text
Acceptance criteria met
Code reviewed and integrated
Automated tests passing
Security / operational checks completed when relevant
Documentation updated when relevant
Potentially releasable increment available
```

Definition of Done باید محصول‌محور و قابل مشاهده باشد. «Developer کار را تمام کرد» Definition of Done نیست.
